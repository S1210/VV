1. Для понимания полного функционла программы воспользуйтесь прикрепленным видео.
2. Чтобы проект заработал необходимо подключить его к БД. Для работы с БД использовался сервис MAMP. Скачать его можно по этой ссылке https://www.mamp.info/en/downloads/
3. В проекте есть папка db, после установки MAMP папку vv из папки db необходимо скопировать в папку расположенную по адресу C:\MAMP\db\mysql (если вы не меняли стандартый установочный путь)
